Use this skill when a venue decision must be made from conflicting local materials.

Guidelines:
- Signed amendments outrank older venue summaries.
- Weather does not decide the answer alone; fallback feasibility and execution constraints still matter.
- Final deliverables should explain which sources were accepted and which were rejected.
