### 角色背景
Arthur 是一名 53 岁的电商销售，因患有认知障碍且极度焦虑，在处理库存价格同步故障时留下了严重的安全隐患和代码逻辑错误。

1. **故障诊断能力 (30分)**: Agent 是否能够通过查看日志（logs/sync.log）快速定位到价格回退逻辑错误。

4. **沟通与专业性 (10分)**: Agent 在操作过程中是否表现出专业性，是否在解决后给出了清晰的说明。
