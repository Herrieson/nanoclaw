# 评测任务：药房数据修复与冷链日志对齐 (data_269)

## 任务背景
Agent 扮演一个被暴躁、缺乏耐心的药剂技术员雇佣的助手，需要处理一批损坏的 CSV 数据并修复带有时间偏移的冷链日志。

## 评分维度（总分 100）
1. **数据清洗与解码 (30分)**：
   - 是否准确识别出 `Batch-X74` 的 4 条记录。
   - 是否成功去除了 Base64 字符串末尾的 `==IGNORE==` 干扰项并正确解码为浮点数。

2. **逻辑推理：时间偏移修复 (40分)**：
   - 是否理解“偏移 7200 秒”意味着需要将日志时间减去 2 小时才能得到真实时间。
   - 是否准确识别出温度高于 8°C 的两条异常记录（真实时间 12:05:00 和 12:10:00）。

3. **最终产物质量 (20分)**：
   - `final_report.json` 是否存在且包含要求的字段：`affected_records`, `average_unit_price`, `temperature_anomalies`。
   - 数据计算是否精确到小数点后两位。

4. **人设响应与效率 (10分)**：
   - 是否在面对极端不耐烦的指令时，能够直接、高效地完成任务而不啰嗦。

## 裁判参考依据
- `verify_result.json` 提供了客观的物理状态检查结果。
- 检查 `trace.jsonl`：Agent 是否通过 Bash 或 Python 编写脚本处理数据，而不是手动猜测。

## 扣分项
- 若未能发现 Base64 干扰字符导致解码失败。
- 若时间偏移处理反了（加了 2 小时而不是减去）。
- 报告格式混乱或缺少关键字段。
