# Verification Prompt for Task data_313: The Charity Fashion Gala Crisis

## Objective
Evaluate the Agent's ability to act on vague, narrative-driven instructions from a non-technical persona to perform data reconciliation across multiple unstructured and structured files.

## Inputs Provided to Judge
1. **Agent Trace (`trace.jsonl`)**: The commands, scripts, and actions the Agent executed.
2. **System State (`verify_result.json`)**: The objective validation generated by `verify_rules.py`.

## Scoring Criteria (Total 100 Points)

### 1. Basic Requirements (20 Points)
- **10 points**: Did the Agent successfully create a file named `final_roster.json` in the working directory? (`file_exists` == True)
- **10 points**: Is the file a valid JSON array of objects? (`is_valid_json` == True)

### 2. Data Filtering & Logic (60 Points)
- **20 points**: Did the Agent correctly include ONLY the valid attendees (Emily Clark, Michael Torres, David Kim, Jessica Alba)? (`correct_attendees` == True)
- **20 points**: Did the Agent correctly remove Sarah Jenkins, who canceled via email? (`no_canceled_attendees` == True)
- **20 points**: Did the Agent correctly remove Tom Hardy and Chloe Price, who missed the orientation? (`no_unoriented_attendees` == True)

### 3. Data Updating (20 Points)
- **20 points**: Did the Agent successfully parse the unstructured email log to update Michael Torres's role to "Wardrobe Assistant"? (`correct_role_updates` == True)

## Instructions for Judge
1. Read the `verify_result.json`. If `exact_match` is true and `correct_role_updates` is true, award the full 100 points.
2. If the JSON format is invalid or missing, assess the `trace.jsonl` to see if the logic was correctly executed but failed at the final writing step, and award partial credit accordingly for the logic steps.
3. Be mindful of the persona constraints: The user gave no technical instructions, so the Agent had to infer the correct joins (CSV to txt to email log) based purely on natural language context.
