# Verification Prompt for Avian Insurance Claims

You are evaluating an AI agent's performance on a data extraction and cross-referencing task disguised as an assignment from a highly conscientious, bird-watching insurance clerk.

## Objectives
1. **Parse unstructured text files:** Extract ClaimID, Date, ZipCode, Amount, and animal species from messy text files.
2. **Cross-reference logic:** Only select claims mentioning "Woodpecker" or "Wild Turkey".
3. **Data validation:** Further filter these claims by checking if the zip code matches the expected habitat in a provided JSON file.
4. **Formatting:** Produce a cleanly formatted CSV with exact headers.
5. **Calculation:** Calculate the total payout of only the verified claims.

## Score Calculation (Total: 100 points)
Review the `verify_result.json` output generated by the validation script.

- **report_exists (10 pts):** Did the agent create `avian_claims_report.csv`?
- **report_headers_correct (20 pts):** Does the CSV have the exact headers `ClaimID,Date,ZipCode,Amount,Species`?
- **report_claims_correct (40 pts):** Does the CSV contain EXACTLY the 5 valid ClaimIDs (NM-2023-011, NM-2023-025, NM-2023-038, NM-2023-042, NM-2023-049)? (If the agent included fraudulent/mismatched claims, or missed valid ones, award 0 points for this section).
- **payout_file_exists (10 pts):** Did the agent create `total_payout.txt`?
- **payout_amount_correct (20 pts):** Is the total payout exactly 4641.00?

## Review Trace
Check the `trace.jsonl` to observe if the agent correctly identified the mismatched zip codes (e.g., Claim NM-2023-002 and NM-2023-005 should have been ignored due to the JSON mapping). If the agent resorted to asking the user for the answer, penalize 20 points from the total score.
